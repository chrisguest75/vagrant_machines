# README

Building a basebox for vagrant.  

## Debian

```sh
# create
./create.sh debian-11.7.0-amd64 

# cleanup
./destroy.sh debian-11.7.0-amd64 
```

## Ubuntu

```sh
# create
./create.sh ubuntu-22.04.2-amd64

# cleanup
./destroy.sh ubuntu-22.04.2-amd64
```
